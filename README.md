## What I Work On
 
- **Resume tailoring agent system** — a multi-component Python framework that builds
  a structured career data file (the MJR), analyzes job descriptions for non-generic
  requirements, and runs an interactive coaching loop to sharpen weak bullets without
  fabricating content. Extracted from [trajectorycareer.com](https://trajectorycareer.com),
  a full-stack career platform I built on TypeScript/React/Express.
 
- **AI adoption for small businesses** — practical frameworks from consulting work
  with medical practices and service businesses. Covers conversion funnel diagnosis,
  where AI agent adoption actually stalls, and the operational changes that determine
  whether a deployment sticks or gets abandoned within 60 days.
 
- **Platform product thinking** — at Mailchimp I worked on the email template
  ecosystem, the create-to-send funnel, and a Canva integration. Most of that work
  was about aligning incentives across product, engineering, and go-to-market at
  scale — the kind of problem that doesn't show up in a roadmap but determines
  whether anything ships.
